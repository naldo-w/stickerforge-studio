WHY DOESN'T MY MESSAGING APP JUST ACCEPT GIFS?

PIXEL-ART ANIMATIONS ARE USUALLY DRAWN AS GIFS, BUT NO MAJOR MESSAGING PLATFORM ACCEPTS GIF STICKERS DIRECTLY — EACH DEMANDS ITS OWN FORMAT, CANVAS AND SIZE LIMIT. CONVERTING BY HAND MEANS JUGGLING SEVERAL COMMAND-LINE TOOLS AND LOSING CRISP PIXELS ALONG THE WAY.

WHAT DOES THIS TOOL DO?

A ONE-STOP TOOL THAT CONVERTS PIXEL-ART GIFS INTO STICKER FORMATS AND TGS (TELEGRAM ANIMATED STICKERS) INTO RASTER FORMATS FOR CROSS-PLATFORM USE. PIXEL-PERFECT NEAREST-NEIGHBOR SCALING, NO GHOSTING, ORIGINAL TIMING PRESERVED. EACH FILE GETS ITS OWN INDEPENDENT PROCESSING PANEL. ALL CONVERSION RUNS LOCALLY IN THE BROWSER — YOUR FILES ARE NEVER UPLOADED. THE ONLY NETWORK REQUESTS ARE FONTS AND, FOR TGS INPUT, A ONE-TIME LOTTIE-WEB FETCH FROM A CDN.

HOW DO I CONVERT A GIF?
  1. ADD — DROP GIFS ANYWHERE OR CLICK ADD GIF. ONE PANEL PER GIF.
  2. REVIEW — EACH FILE GETS A HEADER (NUMBER · NAME · SOURCE INFO · PER-FORMAT STATUS), COLLAPSED BY DEFAULT. CLICK TO EXPAND FOR LIVE ANIMATED PREVIEWS OF EVERY OUTPUT FORMAT — OR SET THE DEFAULT IN SETTINGS → GLOBAL → JOB PANELS.
  3. TUNE — SET DEFAULTS IN SETTINGS; OVERRIDE SCALE, FPS, FRAME RANGE, LOOPS OR QUALITY INSIDE ANY PANEL.
  4. EXPORT — PER FORMAT, PER GIF, OR ALL AT ONCE (EXPORT ALL GIFS · ZIP).
HOW DO I CONVERT A TGS STICKER?
  1. ADD — DROP TGS ANYWHERE OR CLICK ADD TGS. LOTTIE-WEB LOADS FROM CDN ON FIRST USE (NETWORK REQUIRED ONCE).
  2. REVIEW — EACH FILE GETS A COLLAPSIBLE HEADER, THEN ANIMATED SOURCE PREVIEW + AUTO-BUILT OUTPUTS: SIGNAL・512APNG, DISCORD・320APNG, LINE・APNG 270, WEBP (WHATSAPP), PNG 512, PNG CROPPED.
  3. TUNE — SETTINGS → TGS CONVERTER SETTINGS SETS THE DEFAULTS: LOOP COUNT, OUTPUT SIZE (DISCORD・320APNG), QUALITY (WEBP), FRAME SELECTION (PNG). EACH PANEL THEN OVERRIDES PER FILE — AUTO-FIT, FPS AND MAX COLORS ON SIGNAL・512APNG, DISCORD・320APNG AND WEBP.
  4. EXPORT — PER FORMAT IN EACH CELL, PER PANEL ZIP, OR EXPORT ALL TGS · ZIP.

WHAT HAPPENS TO MY PIXELS?

APNG — FULL-FRAME RECONSTRUCTION (GIF DISPOSAL METHODS RESOLVED, NO GHOSTING), TIMING PRESERVED, TRANSPARENT CENTERED CANVAS, PALETTE · DELTA · ADAPTIVE-FILTER COMPRESSION. DISCORD — SAME PIPELINE ON A 320 CANVAS; EACH STICKER NEEDS AN EMOJI TAG ON UPLOAD. PNG — PALETTE + ADAPTIVE FILTER, TRANSPARENT BACKGROUND. TGS — PIXELS AS VECTOR RECTS, ONE PRECOMP REUSED × 2 LAYERS. WEBP — LOSSLESS VP8L DELTA FRAMES + DUPLICATE-FRAME DEDUP (LOSSY VP8 OPTIONAL).

HOW DOES THE TGS CONVERTER WORK?

LOTTIE-WEB RENDERS EACH FRAME TO AN OFF-SCREEN CANVAS AT TARGET SIZE. EVERY ENABLED OUTPUT IS BUILT AS SOON AS THE FILE LANDS, SO EXPORT IS A CACHE READ, AND EACH CELL SHOWS THE ENCODED SIZE — NOT THE SOURCE SIZE. FRAMES ARE ASSEMBLED INTO: APNG (DELTA + PALETTE), DISCORD・320APNG (CONFIGURABLE SIZE), WEBP (VP8/VP8L, LOSSY OR LOSSLESS), PNG (SINGLE FRAME — FIRST / LAST / MID / CUSTOM — WITH OR WITHOUT PADDING). SIGNAL・512APNG WARNS IF ANIMATION EXCEEDS 3 SECONDS.

WHAT HAPPENS IF MY OUTPUT IS TOO BIG?

WHEN AN OUTPUT EXCEEDS ITS PLATFORM BYTE LIMIT, THE FRAME RATE IS STEPPED DOWN FIRST (60 → 30 → 20 → 10 FPS), WHICH COSTS SMOOTHNESS BUT NOT A SINGLE PIXEL. ONLY IF THAT STILL DOES NOT FIT DOES THE PALETTE DROP (256 → 128 → 64 COLORS; FOR LOSSY WEBP, QUALITY DROPS INSTEAD). THE FIRST CANDIDATE AT OR UNDER THE LIMIT WINS. IF NOTHING FITS, THE SMALLEST IS KEPT AND THE CELL WARNS. TURN AUTO-FIT OFF TO SET FPS AND MAX COLORS BY HAND, PER FILE.

WHY IS THERE A 0.4PX OFFSET ON TGS OUTPUT?

ADJACENT VECTOR RECTANGLES CAN SHOW HAIRLINE SEAMS WHEN ANTI-ALIASED BY LOTTIE RENDERERS. THE TGS IS DRAWN AS TWO STACKED COPIES OF THE SAME PRECOMP — THE SECOND NUDGED BY 0.4 PX — SO THE OVERLAP FILLS THE GAPS.
